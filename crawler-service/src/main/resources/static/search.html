Sample of search product and click logs API




How to use

  1. Enter some search words in the text box. (e.g. 'bed', 'Office chair')
  2. Click the [search products] button or Press enter key in the text box.
  3. Search results will be displayed with a table.
  4. Click some links in the [url] colum in the table.
  5. Clicked logs will be created in the click_logs table.
